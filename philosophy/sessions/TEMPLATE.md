# Session Template

Copy this folder structure for each day's exchange.

## Folder: sessions/YYYY-MM-DD/

Create these files:

### plan.md
```
# Session Plan — YYYY-MM-DD

Scenario: [steelman-debate / socratic-dialogue / stoic-reflection / position-defense /
           rhetorical-composition / fundamentals-logic / fundamentals-rhetoric / fundamentals-philosophy]
Ladder: [L1-L5]
Focus: [a move, a thinker, a fallacy, a proposition]

## Warmup (say aloud)
1. A position (claim + warrant) I'll state cold
2. A move I'll reach for (pin-the-term, name-the-warrant, steelman-then-rebut...)
3. A fallacy I'll name on sight

## Session Goal
[What to practice]

## Proposition / Position of the Day
[The claim to defend, or the thinker to reconstruct]

## Micro-Deploy Mission
- Where (a real disagreement, an essay, a post):
- Entry line:
- Goal:
- Exit line:
- Win condition (did the other mind move / did the claim hold?):
```
